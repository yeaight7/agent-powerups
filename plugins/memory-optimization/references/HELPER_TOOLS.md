# Helper Tools

`graphify` is primary memory engine in this bundle.

Helpers:
- `markitdown-file-intake` for PDF / Office docs and other hard-to-read formats
- `defuddle` for standard web pages and articles with noisy HTML chrome

Rules:
- check requirements first with `apx check ...`
- do not auto-install without approval
- use helpers only when they improve readability or graph quality
