# memory-optimization

Graph-backed memory and context optimization workflows with `graphify` first and document conversion helpers second.

Use this extension as a local Agent Powerups bundle. Inspect relevant files under this extension before applying a workflow:

- `skills/` contains reusable workflow instructions.
- `commands/` contains command-first entrypoints.
- `agents/` contains specialist prompts for graph-backed memory planning.
- `references/` contains concise provenance, install, and helper notes.

Do not install external tools, write secrets, or mutate global config unless the user explicitly approves it.
