# agentic-systems

Multi-model orchestration and advisor routing: tri-model review and canonical advisor routing.

Use this extension as a local Agent Powerups bundle. Inspect relevant files under this extension before applying a workflow:

- `skills/` contains reusable skill instructions.
- `agents/` contains reusable agent role prompts when present.
- `commands/` contains slash-command-style prompts when present.
- `templates/` contains reusable project documents when present.

Do not install external tools, enable MCP servers, write secrets, or mutate global config unless the user explicitly approves it.
