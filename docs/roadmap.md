# Roadmap

Scope stays narrow until the repo is public-ready.

## Near Term

- tighten README, docs, catalog, and validation flow
- normalize existing skills, especially tool-dependent ones
- add first public-ready command, hook, MCP, or AGENTS.md examples only after docs and metadata are solid

## Next Assets

- AGENTS.md templates for common project types
- small hook examples with explicit safety guards
- minimal command packs for one or two agent surfaces
- MCP config recipes with clear trust warnings

## Not Planned Yet

- full CLI
- cross-agent orchestration
- hidden auto-installers
- destructive hooks
- broad compatibility claims without concrete assets
