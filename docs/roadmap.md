# Roadmap

Scope stays narrow until the repo is public-ready.

## Near Term

- tighten README, docs, catalog, and validation flow
- normalize existing skills, especially tool-dependent ones
- harden command, hook, workflow, MCP, setup examples, and AGENTS.md assets after docs and metadata stay solid

## Next Assets

- deeper AGENTS.md templates for common project types
- more small hook examples with explicit safety guards
- more command prompts for one or two agent surfaces
- MCP config recipes with clear trust warnings
- platform examples after each target behavior is tested

## Not Planned Yet

- full CLI
- cross-agent orchestration
- hidden auto-installers
- destructive hooks
- broad compatibility claims without concrete assets
